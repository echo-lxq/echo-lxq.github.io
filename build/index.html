gitbook使用介绍

gitbook-template

环境安装

1.nodejs安装

nodejs官方下载地址：https://nodejs.org/zh-cn/

2.gitbook安装

npm install gitbook-cli -g

安装成功后执行

gitbook -V

报错与解决方案 http://wlphp.com/m/?post=369

gitbook模板使用

使用

gitbook模板，克隆到本地执行 npm install 安装插件即可使用（之前需在本地安装gitbook环境）

npm run serve -- 启动web服务，默认端口4000。

npm run build -- 打包部署

目录

assets -- 资源文件目录

pages -- 书章节存放目录

books.js -- gitbook配置文件目录

.bookignore -- gitbook打包忽略文件目录

SUMMARY.md -- 目录文件
